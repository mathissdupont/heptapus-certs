# Hızlı Başlangıç

5 dakikada HeptaCert API'si ile ilk isteğinizi yapın. Bu rehber bir API anahtarı oluşturmayı, kimlik doğrulamayı ve temel CRUD işlemlerini kapsar.

import { Steps, Callout, Tabs } from "nextra/components";

<Steps>

### API Anahtarı Oluşturun

1. [HeptaCert Admin](https://heptacert.com/admin/settings/api) paneline giriş yapın
2. **Ayarlar → API Anahtarları** sayfasına gidin
3. **+ Yeni Anahtar** butonuna tıklayın
4. Anahtara bir isim verin ve gerekli [scope'ları](/api-reference/authentication#scopelar-i̇zinler) seçin
5. Oluşturulan anahtarı kopyalayın — **tek seferlik gösterilir**

Anahtar formatı: `hc_live_xxxxxxxxxxxxxxxxxxxxxxxx`

<Callout type="warning">
API anahtarınızı güvenli saklayın. Git repository'lerine, client-side koda veya log dosyalarına eklemeyin. Ortam değişkeni (`HEPTACERT_API_KEY`) kullanmanız önerilir.
</Callout>

### İlk İsteği Yapın

Etkinlikleri listeleyin:

```bash
curl https://heptacert.com/api/admin/events \
  -H "Authorization: Bearer hc_live_YOUR_KEY"
```

Başarılı yanıt:

```json
[
  {
    "id": 42,
    "name": "Python Summit 2026",
    "event_date": "2026-09-15T09:00:00",
    "certificate_enabled": true,
    "registration_enabled": true,
    "attendee_count": 150
  }
]
```

`401` alıyorsanız anahtarınız hatalı/eksik; `403` alıyorsanız anahtarınızda `events:read` scope'u yok.

### Etkinlik Oluşturun

```bash
curl -X POST https://heptacert.com/api/admin/events \
  -H "Authorization: Bearer hc_live_YOUR_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "name": "React Workshop",
    "template_image_url": "placeholder",
    "event_type": "workshop",
    "certificate_enabled": true,
    "registration_enabled": true,
    "checkin_enabled": true
  }'
```

### Katılımcı Ekleyin

Yukarıdaki yanıttan dönen `id`'yi kullanın:

```bash
curl -X POST https://heptacert.com/api/admin/events/43/attendees \
  -H "Authorization: Bearer hc_live_YOUR_KEY" \
  -H "Content-Type: application/json" \
  -d '{ "first_name": "Ahmet", "last_name": "Yılmaz", "email": "ahmet@example.com" }'
```

</Steps>

## Kod ile Kullanım

<Tabs items={["Python", "Node.js", "cURL + jq"]}>
  <Tabs.Tab>
  ```python
  import os
  import httpx

  API_KEY = os.environ["HEPTACERT_API_KEY"]
  BASE = "https://heptacert.com/api"
  headers = {"Authorization": f"Bearer {API_KEY}"}

  with httpx.Client(base_url=BASE, headers=headers, timeout=15.0) as client:
      # Etkinlikleri listele
      events = client.get("/admin/events").json()
      event_id = events[0]["id"]

      # Katılımcı ekle
      r = client.post(
          f"/admin/events/{event_id}/attendees",
          json={"first_name": "Ahmet", "last_name": "Yılmaz",
                "email": "ahmet@example.com"},
      )
      if r.status_code == 409:
          print("Bu e-posta zaten kayıtlı")
      else:
          r.raise_for_status()
          print("Eklendi:", r.json())
  ```
  </Tabs.Tab>

  <Tabs.Tab>
  ```javascript
  const API_KEY = process.env.HEPTACERT_API_KEY;
  const BASE = "https://heptacert.com/api";
  const headers = {
    Authorization: `Bearer ${API_KEY}`,
    "Content-Type": "application/json",
  };

  // Etkinlikleri listele
  const events = await fetch(`${BASE}/admin/events`, { headers }).then((r) => r.json());
  const eventId = events[0].id;

  // Katılımcı ekle
  const res = await fetch(`${BASE}/admin/events/${eventId}/attendees`, {
    method: "POST",
    headers,
    body: JSON.stringify({
      first_name: "Ahmet", last_name: "Yılmaz", email: "ahmet@example.com",
    }),
  });
  if (res.status === 409) console.log("Zaten kayıtlı");
  else console.log(await res.json());
  ```
  </Tabs.Tab>

  <Tabs.Tab>
  ```bash
  export HEPTACERT_API_KEY=hc_live_YOUR_KEY

  # İlk etkinliğin adını al
  curl -s https://heptacert.com/api/admin/events \
    -H "Authorization: Bearer $HEPTACERT_API_KEY" \
    | jq '.[0].name'

  # Bir etkinliğin katılımcı sayısını al
  curl -s https://heptacert.com/api/admin/events/42 \
    -H "Authorization: Bearer $HEPTACERT_API_KEY" \
    | jq '.attendee_count'
  ```
