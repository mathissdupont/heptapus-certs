# HeptaCert Geliştirici Dokümantasyonu

HeptaCert'in REST API'si, MCP (Model Context Protocol) agent entegrasyonu, CLI aracı ve webhook sistemi için resmi dokümantasyon.

## Neler Yapabilirsiniz?

- **REST API** ile etkinlik, katılımcı, sertifika ve oturum yönetimi
- **MCP Agent** ile Claude veya başka AI asistanlarınıza HeptaCert yetkisi verin
- **CLI (hc)** ile terminal üzerinden tam otomasyon
- **Webhooks** ile sistemlerinizi gerçek zamanlı olarak tetikleyin
