# MCP & AI Agents

HeptaCert offers full integration with AI assistants via the [Model Context Protocol (MCP)](https://modelcontextprotocol.io). Claude, GPT-4, or any MCP-compatible agent — they can manage your events in natural language.

import { Callout, Steps, Tabs } from "nextra/components";

## What is MCP?

MCP allows AI models to securely access external tools and data through a standard protocol. The HeptaCert MCP server includes 35+ tools: event creation, attendee management, certificate issuance, and more.

## Setup with Claude Desktop

<Steps>

### Create an API Key

Go to [Settings → API Keys](https://heptacert.com/admin/settings/api) and create a new key. Required scopes depend on what the agent will do — leave scopes empty for full access.

### Open the Claude Desktop Config File

`~/Library/Application Support/Claude/claude_desktop_config.json` (macOS)
`%APPDATA%\Claude\claude_desktop_config.json` (Windows)

### Add the MCP Server

**Hosted mode (recommended — no local install):**
```json
{
  "mcpServers": {
    "heptacert": {
      "url": "https://heptacert.com/mcp",
      "headers": {
        "Authorization": "Bearer hc_live_YOUR_KEY"
      }
    }
  }
}
```

**Local script mode:**
```json
{
  "mcpServers": {
    "heptacert": {
      "command": "python",
      "args": ["/path/to/heptacert/backend/src/mcp_server.py"],
      "env": {
        "HEPTACERT_API_KEY": "hc_live_YOUR_KEY",
        "HEPTACERT_API_BASE": "https://heptacert.com"
      }
    }
  }
}
```

### Restart Claude Desktop

After restarting, a 🔌 icon will appear in Claude's right panel.

</Steps>

## Example Conversations

```
User: How many attendees checked in today?

Claude: Calling list_events...
        Calling get_event_stats (event_id=42)...

        "Python Summit 2026" has 150 registered attendees,
        127 of whom (84.7%) have checked in.
```

```
User: Create a new event called "React Workshop",
      date October 15 2026, with certificates enabled.

Claude: Creating event...
        ✅ "React Workshop" created (ID: 89).
        A certificate template image URL is needed.
        Would you like to add one?
```

<Callout type="warning">
**Destructive operations require two steps.** When deleting an event, attendee, or certificate, Claude first previews what will be deleted. The operation executes only after you confirm with `confirm=True`.
</Callout>

## Available Tools (35 Tools)

### Events
| Tool | Description | Scope |
|------|-------------|-------|
| `list_events` | List events | `events:read` |
| `get_event` | Single event detail | `events:read` |
| `get_event_stats` | Statistics | `events:read` |
| `create_event` | Create event | `events:write` |
| `update_event` | Update fields | `events:write` |
| `delete_event` | Delete (confirm required) | `events:write` |
| `close_registration` | Close registrations | `events:write` |
| `open_registration` | Reopen registrations | `events:write` |

### Attendees
| Tool | Description | Scope |
|------|-------------|-------|
| `list_attendees` | List attendees | `attendees:read` |
| `add_attendee` | Add attendee | `attendees:write` |
| `update_attendee` | Update fields | `attendees:write` |
| `remove_attendee` | Remove (confirm required) | `attendees:write` |
| `bulk_import_attendees` | CSV bulk import | `attendees:write` |

### Certificates
| Tool | Description | Scope |
|------|-------------|-------|
| `list_certificates` | List | `certificates:read` |
| `issue_certificates` | Issue | `certificates:write` |
| `revoke_certificate` | Revoke (confirm required) | `certificates:write` |

### Sessions, Check-in, LMS, Automation
Sessions (4 tools), check-in (3 tools), LMS (6 tools), automation (2 tools), analytics (3 tools).

## Security

- Every MCP request is authenticated with a Bearer token
- Scopes are enforced at the MCP layer — if a scope is missing the tool won't run
- All write operations are logged to **Agent Logs** (Admin → Settings → Agent Logs)
