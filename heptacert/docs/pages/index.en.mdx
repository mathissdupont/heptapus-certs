# HeptaCert Developer Documentation

Official documentation for HeptaCert's REST API, MCP agent integration, CLI tool, and webhook system.

## What Can You Build?

- **REST API** — manage events, attendees, certificates, and sessions programmatically
- **MCP Agent** — give Claude or any AI assistant full HeptaCert access
- **CLI (hc)** — automate everything from your terminal
- **Webhooks** — trigger your systems in real time

## Base URL

```
https://heptacert.com/api
```

All requests must be made over HTTPS. HTTP requests are rejected.

## Versioning

The current API is stable and unversioned. Breaking changes are communicated via email in advance.
