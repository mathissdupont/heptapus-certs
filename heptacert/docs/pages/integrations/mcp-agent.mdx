# MCP & AI Agents

HeptaCert, [Model Context Protocol (MCP)](https://modelcontextprotocol.io) üzerinden herhangi bir uyumlu AI istemcisine **44 araçlık** tam entegrasyon sunar. MCP açık bir standarttır — yalnızca Claude'a özgü değildir. Bir kez bağladığınızda, AI asistanınız etkinlik oluşturabilir, katılımcı ekleyebilir, sertifika yayımlayabilir ve analitik sorgulayabilir.

## Bağlantı yöntemi: OAuth (önerilen) veya API anahtarı

HeptaCert MCP sunucusuna iki şekilde bağlanabilirsiniz:

1. **OAuth ile giriş (önerilen)** — OAuth destekleyen her istemci (claude.ai, Claude
   Desktop, Cursor, Windsurf, kendi agent'ınız) `https://heptacert.com/mcp` adresini
   eklediğinizde sizi otomatik olarak HeptaCert giriş + izin (consent) ekranına
   yönlendirir. Giriş yapar, vermek istediğiniz yetkileri (scope) onaylarsınız — API
   anahtarı oluşturup kopyalamanıza gerek kalmaz. Her kullanıcı kendi hesabı ve
   yetkileriyle bağlanır; erişimi istediğiniz an **Entegrasyonlar** sayfasından
   kaldırabilirsiniz.

2. **API anahtarı** — Otomasyon/script senaryoları için `Authorization: Bearer
   hc_live_...` başlığıyla doğrudan bağlanın (aşağıdaki config örnekleri).

<Callout>
**OAuth nasıl çalışır?** İstemci token olmadan `/mcp`'ye bağlanır → sunucu
`401 + WWW-Authenticate` ile OAuth keşif adresini bildirir → istemci standart
keşif belgelerini okur (`/.well-known/oauth-authorization-server` ve
`/.well-known/oauth-protected-resource`), gerekirse kendini otomatik kaydeder
(Dynamic Client Registration, RFC 7591) ve Authorization Code + PKCE akışıyla
giriş yaptırır. Herhangi bir standart uyumlu MCP istemcisi için elle ayar gerekmez.
</Callout>

  Kendi agent'ınızı Python'da yazın:

  ```python
  import asyncio
  from mcp.client.streamable_http import streamablehttp_client
  from mcp import ClientSession

  async def main():
      async with streamablehttp_client(
          "https://heptacert.com/mcp",
          headers={"Authorization": "Bearer hc_live_YOUR_KEY"}
      ) as (read, write, _):
          async with ClientSession(read, write) as session:
              await session.initialize()

              # Araçları listele
              tools = await session.list_tools()
              print([t.name for t in tools.tools])

              # Araç çağır
              result = await session.call_tool("list_events", {"limit": 5})
              print(result.content[0].text)

  asyncio.run(main())
  ```

## ChatGPT / OpenAI Actions

ChatGPT, MCP protokolünü desteklemez — bunun yerine OpenAPI tabanlı **Actions** kullanır.

HeptaCert'i ChatGPT'ye bağlamak için:
1. ChatGPT → My GPTs → Create a GPT → Configure → Add Action
2. **Authentication:** API Key, `Authorization: Bearer hc_live_...`
3. **Schema:** `https://heptacert.com/api/openapi.json` adresinden OpenAPI spec'i alın

OpenAI Agents SDK kullanıyorsanız MCP desteği mevcuttur:
```python
from agents.mcp import MCPServerStreamableHttp

server = MCPServerStreamableHttp(
    url="https://heptacert.com/mcp",
    headers={"Authorization": "Bearer hc_live_YOUR_KEY"},
)
```

## Güvenlik

- Her MCP isteği Bearer token ile doğrulanır.
- **Scope'lar MCP katmanında uygulanır.** Anahtarınızda gerekli scope yoksa araç çalışmadan reddedilir. Örneğin `certificates:write` olmadan `issue_certificates` başarısız olur. (bkz. [Scope'lar](/authentication#scopelar-i̇zinler))
- Tüm yazma işlemleri **Agent Logs** sayfasına kaydedilir (Admin → Ayarlar → Agent Logları). `list_agent_logs` aracıyla veya `hc logs list` ile inceleyebilirsiniz.

## Sorun Giderme

| Belirti | Olası neden | Çözüm |
|---------|-------------|-------|
| Araçlar istemcide görünmüyor | Config hatalı veya istemci yeniden başlatılmadı | JSON'u doğrulayın, istemciyi tamamen kapatıp açın |
| `401 Unauthorized` | Anahtar yanlış/iptal edilmiş | Yeni anahtar oluşturup header'ı güncelleyin |
| Bir araç `403` veya "scope" hatası veriyor | Anahtarda gerekli scope yok | Anahtara ilgili scope'u ekleyin |
| Bağlantı kurulamıyor | URL yanlış | `https://heptacert.com/mcp` (sonunda `/` yok) olduğundan emin olun |
| Silme "çalışmıyor" | İlk çağrı yalnızca önizleme | Agent'tan onaylamasını isteyin (`confirm`) |
