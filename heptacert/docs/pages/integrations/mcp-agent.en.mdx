# MCP & AI Agents

HeptaCert integrates with any MCP-compatible AI client via the [Model Context Protocol](https://modelcontextprotocol.io). MCP is an open standard — not specific to Claude.

## Setup

All clients need two things:
- **URL (hosted mode):** `https://heptacert.com/mcp`
- **Authorization header:** `Bearer hc_live_YOUR_KEY`

  Build your own agent in Python:

  ```python
  import asyncio
  from mcp.client.streamable_http import streamablehttp_client
  from mcp import ClientSession

  async def main():
      async with streamablehttp_client(
          "https://heptacert.com/mcp",
          headers={"Authorization": "Bearer hc_live_YOUR_KEY"}
      ) as (read, write, _):
          async with ClientSession(read, write) as session:
              await session.initialize()

              # List available tools
              tools = await session.list_tools()
              print([t.name for t in tools.tools])

              # Call a tool
              result = await session.call_tool("list_events", {"limit": 5})
              print(result.content[0].text)

  asyncio.run(main())
  ```

## ChatGPT / OpenAI Actions

ChatGPT doesn't support MCP — it uses OpenAPI-based **Actions** instead.

To connect HeptaCert to ChatGPT:
1. ChatGPT → My GPTs → Create a GPT → Configure → Add Action
2. **Authentication:** API Key, `Authorization: Bearer hc_live_...`
3. **Schema:** fetch the OpenAPI spec from `https://heptacert.com/api/openapi.json`

If you're using the OpenAI Agents SDK, MCP is supported:
```python
from agents.mcp import MCPServerStreamableHttp

server = MCPServerStreamableHttp(
    url="https://heptacert.com/mcp",
    headers={"Authorization": "Bearer hc_live_YOUR_KEY"},
)
```

## Security

- Every MCP request is authenticated with a Bearer token
- Scopes are enforced at the MCP layer — if a scope is missing the tool won't run
- All write operations are logged to **Agent Logs** (Admin → Settings → Agent Logs)
